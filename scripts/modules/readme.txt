This directory contains the following module scripts.

The following utilities can be found here:
- Get model data (get_model_datasets.py)
- Get observational data (get_obs_datasets.py)
- Functionality for horizontal interoplation/regridding using triangulation (regridding_tools.py)
- Get the PGW future deltas from global parent model (get_parent_anomalies.py)
- Define subregions in the California Current System (get_study_regions.py)
- Set parameters that specify how to calculate the climatology and threshold following the methodology of Hobday et al. (2016) (set_thresh_and_clim_parameters.py)
- Get additional (auxiliary data) for plotting or other applications, such as topography data (get_auxiliary_data.py)
